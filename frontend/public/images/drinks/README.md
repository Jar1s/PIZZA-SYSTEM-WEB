# Drink Images

## Nové fotky napojov

Pre správne zobrazenie napojov na webe, prosím uložte tieto obrázky do tohto priečinka:

1. **Bonaqua sýtená** → `bonaqua-sytena.jpg`
   - Obrázok: Bonaqua sýtená voda (modrá fľaša s modrým uzáverom)

2. **Coca-Cola Classic** → `coca-cola-1l.jpg`
   - Obrázok: Coca-Cola Classic (červená fľaša s červeným uzáverom)

## Formát
- Formát: JPG alebo PNG
- Odporúčaná veľkosť: minimálne 400x600px
- Maximálna veľkosť súboru: 2MB

## Aktualizácia
Kód už je upravený na použitie týchto obrázkov. Po pridaní súborov sa automaticky zobrazia na webe.
